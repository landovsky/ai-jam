## **Epic 4: Collaborative Programming (The "Jam")**

**Goal:** To crowdsource the event agenda through democratic topic proposals and management.

### **4.1 User Story: Topic Suggestion & Sponsorship**

* **Description:** As a registered user, I want to suggest a topic or volunteer to manage one.
* **Motivation:** To ensure the meetup covers the tools I care about and to allow experts to lead the discussion.
* **Acceptance Criteria:**
* [ ] Suggestion form includes: Handle (Short Title), Description, and Expected Outcomes.
* [ ] Any registered user can "Sponsor" a topic to become its Manager (providing content/facilitation).
* [ ] Topic Managers are clearly labeled on the topic card for all users to see.



### **4.2 User Story: Per-Event Weighted Voting**

* **Description:** As a registered user, I want to distribute 3 votes across upcoming topics.
* **Motivation:** To help the community decide which "recipes" are most relevant for the next session.
* **Acceptance Criteria:**
* [ ] Each user receives exactly **3 votes per event**.
* [ ] Users can distribute votes (e.g., 3 on one topic, or 1 on three different topics).
* [ ] Votes do not carry over to future events.
* [ ] There is no automated deadline; voting remains open until an Admin finalizes the agenda.
